# 🏥 Diabetes Prediction AI

## About
AI model that predicts diabetes risk 
from patient lab values.

Built by: Anubhav
Background: Bachelor's in MLT

## Model Details
- Algorithm: Random Forest
- Dataset: 768 patients
- Accuracy: 77.27%
- Cross Validated: 75.67%

## Lab Values Used
- Glucose
- Blood Pressure
- BMI
- Insulin
- Skin Thickness
- Age
- Pregnancies
- Diabetes Pedigree

## Limitations
- Trained on female patients only
- American dataset
- Needs Indian patient data
